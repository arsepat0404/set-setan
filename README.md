# set-setan

Bagian dari ekosistem [Arsepat Game Hub](https://arsepat-game.web.id).

## URL

https://set-setan.web.id

## Stack

TanStack Start + Cloudflare Workers + Supabase

## Development

```bash
npm install
npm run dev
```

## Deploy

Otomatis via Cloudflare setiap push ke branch `main`.

## Catatan migrasi

Dibersihkan menggunakan Lovable Migrator v5.
